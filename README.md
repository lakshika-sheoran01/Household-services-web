# Household-services-web
This project is about the application where the customer can book any professional for his household services. It offers a user-friendly interface that simplifies the problem of managing various services that includes painting, plumbing etc. Technologies Used:Flask-SQLAlchemy is used for defining and interacting with database models. Matplotlib is used for generating statistical plots and visualizations. BOOTSTRAP- is used for frontend pages for designing and creativity of website pages.HTML framework for building user interface.
